# Design Guidelines for Osinter x - OSINT Portfolio Website

## Brand Identity
- **Name:** Osinter x
- **Primary Color:** Deep Black (#050505) - Background
- **Accent Color:** Neon/Parrot Green (#39ff14) - Text highlights, borders, glowing effects
- **Aesthetic:** Cyberpunk, Hacker, Terminal-style, High-tech, Mysterious

## Typography
- **Headers:** Monospace font (Fira Code or Courier New) for coding vibe
- **Logo:** Monospace font with glowing green effect
- **Body Text:** Clean, readable font that complements the terminal aesthetic

## Visual Effects & Styling

### Glowing Effects
- Extensive use of glowing box shadows on buttons and cards using the neon green (#39ff14)
- Logo should have a green glow effect
- Card borders glow on hover

### Special Effects
- Glitch effect on main hero title
- Typewriter effect for cycling text in hero section
- Subtle Matrix rain or grid overlay with low opacity on black background
- Terminal-like design elements throughout

## Layout Structure

### Navbar
- Logo: "Osinter x" (monospace, glowing green)
- Navigation links: Home, Methodology, Tools, Contact
- Clean, minimal design

### Hero Section
- Center-aligned layout
- Headline: "Uncovering the Hidden Truths" or "Mastering Open Source Intelligence"
- Typewriter effect sub-headline cycling through:
  - "Digital Forensics"
  - "Data Analysis"
  - "Cyber Investigation"
- CTA Button: "Initiate Search" - Hollow style with green glow effect

### About/Skills Section
- Grid layout displaying skill cards
- Skills include: Geolocation, Social Media Analysis, IP Tracing
- Card design: Dark grey background with green border
- Green glow effect on hover

### Contact Section
- Terminal-like box design
- Text: "Secure Line Established"
- Prominent Telegram button with icon
- Link: https://t.me/SC0FiiElD (opens in new tab)

## Interaction & Animation
- Framer Motion entrance animations for all sections
- Smooth hover effects on interactive elements
- Glowing button states
- Card hover interactions with border glow

## Responsive Design
- Fully mobile-friendly
- Mobile-first approach
- Ensure all glowing effects and animations work smoothly on mobile devices

## Component Styling Principles
- Cards: Dark grey backgrounds with thin green borders, glow on hover
- Buttons: Hollow/outline style with green borders and glow effects
- Inputs (if any): Terminal-style with green focus states
- All interactive elements should have the cyberpunk aesthetic with green accents

## Images
No hero background image required. The design relies on the Matrix-style grid overlay and deep black background with glowing green accents for visual impact.
